# `comet` — 0603 → 0402 resistor migration

Worksheet for migrating the resistors in the **`comet`** schematic from 0603 to
0402 packages. Generated by Henley: current parts read live from the Fusion
Electronics design (each part's `LCSC` attribute), candidate 0402 replacements
verified against the JLCPCB OpenAPI (`henley detail`) on **2026-06-28**.

Every current 0603 resistor here is a JLCPCB **Extended** (`expand`) part; the
proposed 0402 replacements are all **Basic** (`base`) parts — so this migration
both shrinks the footprint *and* removes the per-extended-part assembly fee.

## Swap table

| Designators | Qty | Value | Current (0603) | → New (0402) | New MPN / mfr | Notes |
|---|---|---|---|---|---|---|
| R1, R2, R4, R5, R7, R9 | 6 | 22kΩ | `C2907015` (Extended, ±1%) | **`C25768`** (Basic, ±1%) | 0402WGF2202TCE / UNI-ROYAL | drop-in ✅ |
| R8 | 1 | 10kΩ | `C2906982` (Extended, ±1%) | **`C25744`** (Basic, ±1%) | 0402WGF1002TCE / UNI-ROYAL | drop-in ✅ |
| R13 | 1 | 220Ω | `C114683` (Extended, ±5%) | **`C25091`** (Basic, ±1%) | 0402WGF2200TCE / UNI-ROYAL | ⚠️ power — see below; tolerance upgrade 5%→1% |
| R6 | 1 | 220Ω ×4 | `C29719` (Extended, 0603×4 array) | *(no 0402 drop-in)* | — | resistor **array** — not a 1:1 0402 swap; see below |
| R3 | 1 | 309kΩ | `C2999532` (Extended, ±1%) | *(no Basic 0402)* | — | uncommon value; stays Extended — see below |

All proposed parts are thick-film, ±100ppm/℃, and were in deep stock when
checked (C25768 ≈ 616k, C25744 ≈ 1.29M, C25091 ≈ 479k). Each is a fraction of a
cent in unit price.

## Electrical caveats (inherent to the 0603 → 0402 shrink)

0402 thick-film resistors here are rated **62.5mW (1/16W)** and **50V max**,
versus **100mW / 75V** for the 0603 parts. Tolerance/tempco are unchanged
(±1%, ±100ppm/℃).

- **22kΩ and 10kΩ (R1/R2/R4/R5/R7/R9, R8):** dissipation is ~1mW even on a 5V
  rail — far under 62.5mW. **Safe.** ✅
- **309kΩ (R3):** negligible power. The only blocker is sourcing (below). ✅ on power.
- **⚠️ 220Ω (R13, and R6's array):** these are the ones to check. 220Ω across
  3.3V dissipates ~50mW (OK); across **5V it is ~114mW, which exceeds the 0402
  62.5mW rating.** Before moving R13/R6 to 0402, confirm the actual voltage /
  current they carry (e.g. LED series, gate, or bleed resistor). If they sit on
  5V or carry real current, keep them at 0603 (or use a higher-rated 0402).

## R6 — resistor array (special case)

R6 is `C29719` = `4D03WGJ0221T5E`, a **4-element 220Ω resistor array** in a
`0603×4` package, not a single chip. There is no 0402 equivalent of a 4-resistor
array. Options:
- **Keep R6 as the array** (simplest), or
- **Replace with 4× discrete 0402 220Ω** (`C25091`) — this is a schematic/layout
  change (4 parts + nets instead of one), not a part-number swap, and is subject
  to the same 220Ω power caveat above.

## R3 — 309kΩ (no Basic 0402)

309kΩ is an uncommon E96 value; JLCPCB has **no Basic 0402 part** at this value
(searched LCSC/JLC, June 2026). A 0402 309kΩ ±1% (e.g. FOJAN `FRC0402F3093TS` /
UNI-ROYAL `0402WGF3093TCE`) would still be an **Extended** part — so migrating R3
saves footprint but **not** the extended-part fee. Look the exact code up
manually and verify with `henley detail <code>` before committing. If a Basic
part matters more than the exact value, a nearby Basic 0402 value (e.g. 300kΩ or
330kΩ) could substitute — but that's a design decision, not a like-for-like swap.

## Applying a swap

For each resistor, update its Fusion part attributes and footprint:

- `LCSC` → new code (e.g. `C25768`)
- `MPN` → new MPN (e.g. `0402WGF2202TCE`)
- `MANUFACTURER` → `UNI-ROYAL`
- package/footprint → 0402

(When Fusion Electronics gains write access, Henley will be able to push these
attribute changes back into the schematic directly; until then, apply them in
Fusion.)
